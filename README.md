# Mission Control

Mission Control ist eine lokale AI-Agentenplattform für Unternehmen.

## Technologien

- Open WebUI
- Ollama
- Docker
- PostgreSQL
- Qdrant
- n8n

## Ziel

Mission Control dient als zentrale Plattform für spezialisierte AI-Agenten, die Geschäftsprozesse unterstützen und automatisieren.
