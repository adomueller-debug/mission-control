# Mission Control Architektur

## Komponenten

- Open WebUI
- Ollama
- PostgreSQL
- Qdrant
- n8n

## Agenten

- BOSS
- FORGE
- AURA

## Ziel

Mission Control ist eine modulare AI-Plattform, bei der jeder Agent eine klar definierte Rolle hat und über gemeinsame Dienste (Memory, Datenbank, Workflows) zusammenarbeitet.
